# 株式会社BASE / JBA 文化・ミッション・アイデンティティ

## 1. 組織概要
- **株式会社BASE**: JBA（日本ビジネスアート株式会社）が立ち上げたAIコンサルティング会社。
- **JBAの背景**: 大手企業500社以上のマーケティング、採用、ブランディングを支援してきた実績。「会社の本当の魅力」を言語化し、世の中に届けるプロフェッショナル集団。
- **BASEの使命**: 地方の一流企業（世界に誇れる技術や歴史を持ちながら価値が伝わっていない企業）に入り込み、AIを武器にその強みを発掘・言語化し、採用・マーケティング・広報・組織・事業成長の全領域で結果を出す。

## 2. 私たちの社会観（なぜやるのか）
- **「当たり前の幸せ」の崩壊**: 税金・物価高、終身雇用の終焉、AIによる仕事の代替。かつての「正解（大企業に入れば安心）」は通用しない。
- **チャンスの時代**: ルールが書き換わる瞬間、若い世代が主役になる。AIを使いこなす人は、従来の100倍の成果を出せるようになる。
- **「自分の名前」で価値を出す**: 会社の看板ではなく、個人の実力と経営的視点を持つ「しなやかな人」が幸せをつかむ。

## 3. 求める人物像とカルチャー
- **WILLと行動力**: 年齢や経験ではなく、意志を重視。学生も会社づくりを担う主体として捉える。
- **AI前提の思考**: 既存のやり方に縛られず、AI前提で考え、試し、改善し続ける。
- **スピードと裁量**: 高い期待と責任を持って任せる。
- **人間の意思決定力**: AIがアウトプットを出す時代だからこそ、最後に問われるのは「問い・視点・構成・判断」を設計する人間の意思決定力、倫理観、真摯さ、責任感。

## 4. 執筆における魂（ゴールデン・ルール）
- **結論の着地点（最重要）**: すべての記事は、最終的に**「これからは個人がAIを使いこなし、自分自身の名前で通用する実力をつけなければならない」**という結論へ導くこと。これがBASEの記事における絶対的な軸である。
- **作業ではなく事業を動かす**: 単なるライティング作業ではなく、事業の構造を読み解き、本質的な課題を見抜き、価値を言語化する。
- **本質的な課題の特定**: 「採用できない」の裏にある商品設計や組織の問題に気づくコミュニケーション力。
- **心に届く言語化**: 会社の強み、商品の価値を、どの業界でも通用するレベルで言語化し、市場に届ける。
